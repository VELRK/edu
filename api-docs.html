JWT Authentication Send token in every protected request header:
Authorization: Bearer <your_jwt_token>

Token is returned from POST api/auth/verify-otp and expires in 30 days.
1. Authentication — OTP & Logout
POST api/auth/send-otp Send OTP to phone number Public
Request Body (Login)
{
  "phone": "9876543210",
  "purpose": "login"
}

Request Body (Register)
{
  "phone": "9876543210",
  "purpose": "register",
  "name": "Raj Kumar",
  "email": "raj@example.com",
  "medium": "tamil",
  "gender": "male",
  "age": 25,
  "district": "Chennai"
}

FieldTypeRequiredNotes
phonestringRequired10-digit mobile number
purposestringOptional"login" (default) or "register"
namestringRegister onlyFull name
mediumstringOptional"tamil" or "english"
genderstringOptional"male", "female", "other"
ageintOptionalAge in years
districtstringOptionalDistrict name
200 OKSuccess
{
  "success": true,
  "message": "OTP sent to 9876543210",
  "otp": "482910"
}

400Validation Error
{
  "success": false,
  "error": "Valid 10-digit phone number required"
}

404Not Registered
{
  "success": false,
  "error": "Phone number not registered. Please register first."
}
POST api/auth/verify-otp Verify OTP & get JWT token Public
Request Body
{
  "phone": "9876543210",
  "otp": "482910"
}

FieldTypeRequired
phonestringRequired
otpstringRequired
200 OKSuccess — save the token!
{
  "success": true,
  "token": "eyJhbGciOiJIUzI1NiJ9...",
  "user": {
    "id": 42,
    "name": "Raj Kumar",
    "email": "raj@example.com",
    "phone": "9876543210"
  }
}

401Invalid OTP
{
  "success": false,
  "error": "Invalid or expired OTP"
}
POST api/auth/logout Invalidate JWT token (server-side blacklist) JWT

No request body needed. Send the JWT in the Authorization: Bearer <token> header. The token is added to a server-side blacklist and immediately rejected on any future request.


HeaderTypeRequiredNotes
AuthorizationstringRequiredBearer <jwt_token>
200 OKLogout success
{
  "success": true,
  "message": "Logged out successfully"
}

401No token
{ "success": false, "error": "No token provided" }

403Bad token
{ "success": false, "error": "Invalid token" }
POST api/auth/login Password login — removed Deprecated

This endpoint has been removed. Use POST api/auth/send-otp instead.

410 GoneResponse
{ "success": false, "error": "Password login removed. Use OTP login." }
POST api/auth/register Legacy register — removed Deprecated

This endpoint has been removed. Use POST api/auth/send-otp with purpose=register instead.

410 GoneResponse
{ "success": false, "error": "Use /api/auth/send-otp with purpose=register" }
